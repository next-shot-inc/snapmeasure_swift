Interpretation Tools

Once a picture has been taken, the interpretation screen appears with its associated tools. The image is typically of higher resolution than the device screeen so you can zoom-in/zoom-out or pan with two fingers. You can go back to the main screen using the "Done" or "Cancel" buttons.

It is recommended to digitize while zoomed-in not only to be more precise with the interpretation but also to optimize the rendering of the image which is typically many times the resolution of the screen.

Below are the different digitization tools:
Draw/Edit mode: Draw or edit a boundary (top, unconformity, fault) with one finger.
 
Pick mode: Select a line to edit it, change its name or color or type.
New line: Create a new line (new name).
Paint Layers: Toggle on/off the coloring of the outcrop layers (The lines bounding the layers must touch the borders or other lines).
Erase mode: Delete interpretation below (boundary, facies, text)
Measure mode: Measure distance between two points (once the scale is defined)
Scale mode: Define the real distance between two points and therefore the picture scale. Enter the reference size in the text field and draw a line of that same distance on the picture.
Facies mode: Interpret the facies/sedimentary style of an interval. Create a column of facies interpretation (pseudo-well) by snapping to interval above. The first interval width will define the width of the column.
Text mode: Draw a rectangle and type text to annotate the image.
Dip/Azimuth mode: Measure dip/Azimuth with device and locate the measurement on the image. The point snapped to a given line will have the same color as the line. The apparent dip will be shown on the picture.
Define feature: Measure height and width of a feature and define the feature type (channel, lobe, etc.). These will be used for computing features statistics.

To export interpretations inside a GOCAD model, use the "Save" button at the top and the export options will be available:
Mail: Email the image and the interpretation in GOCAD format. The image will be embedded in a Voxet, facies columns in a Well curve, lines will be PLines, and dip/azimuth points will be VSets with a normal.
Warning:An image without a scale defined cannot be exported.
Export: Save the current interpretation in GOCAD format (as above). The image file and the GOCAD file can then be found in the Files app available since IOS 11.
